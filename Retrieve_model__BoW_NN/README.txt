Pour entraîner le modèle, il faut :

- le corpus des informations sur les films : intents.json
- le fichier Python train_chatbot.py
- les fichiers classes.pkl et words.pkl

Le corpus a été créé à partir du fichier IMDb_movies.csv et le notebook Chatbot_corpus.ipnyb

On enregistre le modèle dans le fichier chatbot_model.h5.

Le chatbot est créé dans le fichier Python chatgui.py

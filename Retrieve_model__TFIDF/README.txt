Pour lancer le chatbot sur les films allocine étudiés, il faut exécuter le fichier Python main.py

- Les films et leurs informations sont disponibles dans le fichier csv allocine_parser_results.csv
- Le corpus des questions est contenu dans corpus_questions.json

Le fichier allocine_parser.py est le code ayant permis de générer le fichier csv des films allociné.
ATTENTION : le site a été modifié et certaines informations ne sont plus disponibles dorénavant !

Les autres fichiers Python permettent l'exécution de main.py
On y retrouve les étapes de preprocessing, de cconception des matrices TF-IDF et de calcul de cos-similarité.
